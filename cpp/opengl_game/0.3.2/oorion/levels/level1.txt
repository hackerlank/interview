# Создаем игрока - описание параметров функции, по порядку
# --- CreatePlayer()
# - Здоровье
# - Патроны
# - Очки
# - Имя текстуры
# - Координата X
# - Координата Y
# - Ширина кадра текстуры
# - Высота кадра текстуры
# - Ширина картинки (без учета округления до 2^n)
# - Высота картинки (без учета округления до 2^n)
# - Первый кадр стрельбы N1 (всего три типа стрельбы)
# - Количество кадров в анимации первой стрельбы
# - Первый кадр второй стрельбы
# - Количество кадров анимации второй стрельбы
# - Третья стрельба
# - Третья стрельба (то же самое)
# - Первый кадр анимации 'Stand'
# - Количество кадров в этой анимации
# - Первый кадр бега
# - Количество кадров бега
# - Анимация сидя
# - Количество кадров
# - Анимация idle
# - Количество кадров
# - Анимация talk
# - Количество кадров
# - Анимация jump
# - Количество кадров
# - Кадр, когда игрок начинает падать вниз
# - Начальная анимация, допустимые значения:
# -		animStand - стоять
# -		animWalk
# -		animSit
# -		animIdle - idle, и т.д.
# -		animTalk
# -		animJump
# -		animShoot1
# -		animShoot2
# -		animShoot3

# Создание бонуса - описание параметров функции, по порядку
# --- CreatePowerup()
# - Тип бонуса, допустимые значения:
# -		pwrHealth - здоровье
# -		pwrAmmo - патроны
# -		pwrScore - очки
# - Количество единиц бонуса ( прибавка)
# - Координата X 
# - Координата Y

CreatePowerup(pwrAmmo, 20, 150, 400);
CreatePowerup(pwrScore, 30, 600, 400);
